技术栈：Unity 2022、C#、UGUI、DOtween、LitJson、对象池、协程异步、事件中心、场景状态机

简介：复刻《保卫萝卜》完整塔防玩法

负责模块：

1. UI方向制作：横向循环列表、UI动画、采用中介者模式统一管理UI面板，通过工厂加对象池实现面板复用

2. 防御塔核心交互逻辑：实现空地点击造塔、选中塔升级 / 出售、自动锁定范围内怪物、子弹弹道发射逻辑；区分单体伤害、范围 AOE、减速控制三类防御塔，各自封装独立攻击逻辑

3. 金钱经济系统：击杀怪物产出金币，造塔、升级消耗金币，金币不足时禁用建造按钮，UGUI 实时刷新金币 UI 数值

4. 地图制作与交互：配备地图制作工具，便于策划制作地图，实现可交互地图障碍系统，防御塔可锁定障碍攻击，清除后解锁新造塔点位

5. 全局游戏流程状态机：区分准备波次、战斗中、胜利结算、失败重开4 种状态，通过事件中心解耦 UI、怪物、塔、音效模块，降低代码耦合

6. 优化方向：子弹采用无实体射线检测优化性能，大范围群怪场景下渲染开销大幅降低，对象池管理怪物实体，复用怪物，减少频繁实例化与销毁产生的 GC，解决大波怪物同时生成时帧率暴跌问题
